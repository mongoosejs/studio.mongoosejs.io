Frequently asked questions

What is Mongoose Studio?
Mongoose Studio is a Mongoose-centric MongoDB UI built by members of the Mongoose team. Mongoose Studio is a Node.js app that supports integrations with Express, Netlify, and Vercel, meaning you can run Mongoose Studio as part of your Node.js application.
Do I need to use Mongoose.js to use Mongoose Studio?
Yes. Mongoose Studio uses your Mongoose connections and schemas to connect to the database and infer the structure of your data.
Is Mongoose Studio compatible with all MongoDB databases?
Yes. If your application can connect to a MongoDB database, then you can use Mongoose Studio to browse that database. Mongoose Studio supports self-hosted MongoDB instances, as well as managed solutions like MongoDB Atlas.
Does Using Mongoose Studio require coding experience?
Not necessarily. Non-technical users can still explore data and view dashboards. However, Mongoose Studio is primarily meant to be a tool for developers.
Is Mongoose Studio secure?
Yes. Mongoose Studio Pro supports GitHub OAuth, which helps your team securely access your data without relying on MongoDB connection strings. Mongoose Studio also supports role-based access control to further protect your data.

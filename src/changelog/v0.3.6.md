## Chat Share Button Moved to Sidebar

The **share chat** button has moved from its previous position in the top-right corner of the chat area to the sidebar footer.
This keeps the chat area cleaner and groups the share action alongside other sidebar controls, ensuring that it doesn't hover over any chat messages.

<img src="https://res.cloudinary.com/drfhhq8wu/image/upload/v1774984919/Screenshot_from_2026-03-31_15-20-50_an5igy.png">

## Router-Link for JSON Document Open

The "Open this Document" action in the JSON document viewer now uses a proper `<router-link>` instead of a click handler.
This means the link opens in a new tab, can be middle-clicked, and behaves like a standard browser link — so you can more easily open documents in new tabs.

<img src="https://res.cloudinary.com/drfhhq8wu/image/upload/v1774985006/Screenshot_from_2026-03-31_15-23-18_tfhwys.png">

## Discord and GitHub Help Links

When Mongoose Studio can't connect to your MongoDB instance, the connection error screen now shows links to **ask in Discord** and **open a GitHub issue**.
If you're stuck on a connection problem, you can get help directly from the error state without searching for support channels.

## Editting Dashboards via Chat

The dashboard view now has a "Start Chat" option that starts a new chat based on that dashboard.

<img src="https://i.imgur.com/AJ1chtF.png">

You can use this to easily edit existing dashboards, from basic tasks like updating colors to more sophisticated tasks like adding new elements.
Iterate on your dashboard in the chat tab, and then click "Overwrite Dashboard" to update your dashboard.

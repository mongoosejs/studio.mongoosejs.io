## New Array UI

No more plain JSON arrays in the document detail view.
Arrays are now collapsed to the first 3 items by default, and can be expanded to view all items.
For large arrays, see a preview up front, plus a one-click option to display every item.

The UI also has index badges for each item, making it easy to identify and navigate through the array; and no more curly braces between items cluttering the view.
